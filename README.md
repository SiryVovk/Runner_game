# Runner_game
1 Опис проєкту 
 Проєкт ранер з керуванням свайпами. Швидкість аватара постійно збільшується. При поразці запускається анімація смерті й зявляється напис.
Реалізувати реєстрацію і авторизацію у грі.
Архітектура проєкту 
 1. PlayerController
	Модуль відповідає за управління персонажем
	Взаємодіє з GameManager для керування станом гри. Взаємодіє з AnimationController для керування анімаціями гравця.
 2. GameManager
	Модуль відповідає за стан гри.
	Взаємодіє з UIManager та LevelGenerator для управвління грою 
 3. LevelGenerator
	Відповідає за генерацію рівня і створення перешкод
Взаємодіє з GameManager для управління генерацією перешкод
 4. UIManager
	Відповідає за управління інтерфейсом. Нарахування очок за пройдену відстань і за підбір монет.
	Взаємодія з GameManager для відображення стану гри
 5. AnimationController
	Відповідає за анімацію в грі
	Взаємодіє з PlayerController та GameManager для відображення анімацій
 6.Data
	Відповідає за зберігання данних результатів проходження рівнів.
	Взаємодіє з UIManager для відображення результатів гри
